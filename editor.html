What's this?

Instead of linking to an image you can upload the image, copy the code and then paste it into the ![Image Name](paste here...). This stores the image directly into the file instead of the image being called from an external source. If the markdown file is encrypted, the external link is not, so use this solution to encrypt the image in the file. If the file is large, the code, encoded as base64, can get quite big so it is best to reduce the image size if possible.
